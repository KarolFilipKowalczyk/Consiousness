\chapter*{Abstract}

This work presents a unified computational theory of consciousness that integrates insights from Integrated Information Theory \cite{tononi2004,tononi2016}, Global Workspace Theory \cite{baars1988,baars1997,dehaene2001}, Attention Schema Theory \cite{graziano2013,graziano2019}, and quantum consciousness approaches \cite{penrose1989,hameroff1996} within a novel framework based on finite machine hierarchies. 

% Note: We use entropic scaling I(n) = κ n log n throughout
% This provides super-linear but sub-exponential growth,
% avoiding unrealistic exponential resource requirements

The central thesis proposes that consciousness emerges from the collapse of parallel computational explorations across a hierarchy of finite-state machines with entropically growing resources ($M_1, M_2, M_3, \ldots, M_n$ where $M_n$ has effective information capacity $I(n) = \kappa n \log n$ bits). The hierarchy exhibits a generalized non-uniform structure where effective levels follow $M_n \subseteq M_{n+f(n)}$ with variable resource gaps $f(n) \geq 1$, representing realistic cognitive resource allocation where transitions between processing levels involve variable jumps. While real computational time includes all parallel attempts, backtracks, and time-reversals, subjective conscious experience perceives only the successful computational path—a smooth, continuous temporal flow that never experiences the failed branches.

This framework addresses the hard problem of consciousness \cite{chalmers1995,chalmers1996} by explaining why there is ``something it is like'' to be an information-processing system: consciousness is the internal phenomenology of resource-constrained computational collapse, where the selector mechanism (analogous to quantum measurement but implementable classically) chooses which machine $M_n$ to deploy. The non-computability of optimal selection, related to Kolmogorov complexity \cite{kolmogorov1965}, provides a natural account of agency and free will.

The theory makes testable predictions about neural correlates of consciousness \cite{koch2016}, explains why consciousness is decidable at each resource level yet computationally universal in the limit, and provides a mechanistic account of temporal phenomenology that existing theories lack. By treating computational power itself as an explorable dimension parallel to problem space, and time-reversal as a resource invisible to consciousness, this framework bridges computational complexity theory \cite{turing1936,sipser2012}, neuroscience, and phenomenology \cite{husserl1991} in a comprehensive synthesis.
